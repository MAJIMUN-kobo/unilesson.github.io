Overview

C#を使用して制作したハイスピード3Dアクションです。物理演算を用いた爽快な挙動が特徴で、プレイヤーはサイバーパンクな都市を舞台に障害物を避けながらゴールを目指します。

開発のテーマとして「いかにプレイヤーが気持ちよく操作できるか」に注力し、カメラワークやジャンプの慣性などを細かく調整しました。

Features

  • 物理演算を使った滑らかで直感的な操作感
  • ポストプロセッシングによるサイバーパンクの表現
  • オリジナルC#スクリプトによるダイナミックなカメラ追従

Info

  • Engine: Unity 2022.3
  • Language: C#
  • Genre: 3D Action
  • Platform: WebGL (unityroom)
  • Release: 2026.02.10
